# Security Policy

## Public-Safe Use

Do not put secrets, credentials, tokens, private customer records, employee records, health information, legal matter details, payment data, production incident details, or account data in this repository or in screen examples.

The included scripts are local-only and use the Python standard library. They do not call external APIs or require network access.

## Reporting Issues

For public repository issues, report problems without sensitive details. Use synthetic examples or minimal redacted reproductions.

If a problem involves private data, credentials, or a real organization, do not paste that material into an issue, pull request, commit, rendered HTML file, or shared example.

## Scope

This project validates and renders a simple public-safe screen format. It does not perform vulnerability scanning, privacy review, legal review, compliance review, procurement review, or production security approval.
