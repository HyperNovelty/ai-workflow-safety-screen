# Public Safety Boundary

This repository is intended to be safe to inspect, fork, and run locally without exposing private systems or data.

## Included

- Synthetic example data.
- A small JSON schema.
- Python standard-library validation and rendering scripts.
- Documentation for early AI workflow screening.
- Local-only commands.

## Excluded

- Real customer, employee, patient, student, applicant, account, or client data.
- Credentials, secrets, tokens, private keys, API keys, or local account configuration.
- Private WSL paths and machine-specific dashboard internals.
- Build artifacts, virtual environments, pycache, pytest cache, product-package exports, generated candidate packets, and private candidate data.
- External API calls, hosted services, deployment scripts, scheduling, telemetry, and publishing flows.

## Advice Boundary

This project does not provide legal, compliance, procurement, cybersecurity, medical, financial, employment, tax, insurance, or investment advice. It is not an approval mechanism. It does not certify that a workflow is safe, lawful, compliant, secure, or ready for production.

## Data Boundary

Do not use screen files to collect sensitive data. If a real workflow is being discussed, reduce it to public-safe language before writing an example. Replace identifying details with roles, categories, and synthetic descriptions.
